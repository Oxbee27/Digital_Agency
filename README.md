# SquareUp - Digital Agency Website

A modern, responsive digital agency website built with **HTML**, **CSS**, and **Tailwind CSS**. SquareUp is a digital product studio that specializes in design, engineering, and project management services.

## 🚀 Features

- **Responsive Design** - Mobile-first approach with Tailwind CSS grid system
- **Modern UI** - Dark theme with lime green accents (#9eff00)
- **Service Showcase** - Highlights Design, Engineering, and Project Management services
- **Testimonials** - Client success stories and case studies
- **FAQ Section** - Interactive accordion-style FAQ component
- **Contact Form** - Budget range slider and multi-field contact form
- **Icon Library** - Integrated Boxicons for visual enhancements

## 📦 Installation

### Prerequisites
- Node.js (v14 or higher)
- npm (comes with Node.js)

### Setup

1. **Clone the repository**
   ```bash
   git clone git@github.com:Oxbee27/Digital_Agency.git
   cd Digital_Agency
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **View the website**
   - Open `index.html` directly in your browser, or
   - Use a local development server (e.g., Live Server extension in VS Code)

## 🎨 Tailwind CSS

The project uses Tailwind CSS for styling. If you need to modify styles:

1. Edit your CSS in the source files
2. Run the Tailwind CLI to compile:
   ```bash
   npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
   ```
